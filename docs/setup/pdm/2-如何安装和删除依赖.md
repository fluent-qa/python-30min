PDM（Python Dependency Manager）是一个现代的Python包和依赖管理工具。以下是如何在项目中使用PDM安装和删除依赖的简要指南。

## 安装依赖

1. **安装PDM**：
   - 首先需要安装PDM，可以使用以下命令：
     ```bash
     pip install pdm
     ```

2. **初始化项目**：
   - 在项目根目录下运行以下命令来初始化项目：
     ```bash
     pdm init
     ```
   - 这将创建一个`pyproject.toml`文件，用于存储项目的配置信息。

3. **添加依赖**：
   - 使用以下命令添加依赖包：
     ```bash
     pdm add <package_name>
     ```
   - 例如，添加`requests`包：
     ```bash
     pdm add requests
     ```
   - PDM会自动更新`pyproject.toml`和`pdm.lock`文件以锁定依赖版本。

## 删除依赖

1. **删除依赖**：
   - 使用以下命令删除依赖包：
     ```bash
     pdm remove <package_name>
     ```
   - 例如，删除`requests`包：
     ```bash
     pdm remove requests
     ```
   - PDM会更新`pyproject.toml`和`pdm.lock`文件，并同步工作集以移除指定的包。

## 示例

### 安装依赖

```bash
# 安装PDM
pip install pdm

# 初始化项目
pdm init

# 添加requests包
pdm add requests
```

### 删除依赖

```bash
# 删除requests包
pdm remove requests
```
